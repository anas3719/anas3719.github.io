# صفحة بورتفوليو مودل

صفحة شخصية عربية جاهزة للتقديم على الإعلانات والتعاونات التجارية.

## تعديل البيانات

- الاسم ومعلومات التواصل محدثة لأنس عمر.
- لتغيير البريد، الانستغرام، أو الواتساب لاحقًا عدل قسم `contact` في `index.html`.
- عدلي المدينة، المجالات، والمقاسات في قسم `details`.

## إضافة الصور

أنشئي مجلد باسم `assets` بجانب الملفات، ثم أضيفي صورك بهذه الأسماء:

- `hero.jpg`
- `photo-1.jpg`
- `photo-2.jpg`
- `photo-3.jpg`
- `photo-4.jpg`

الصفحة تعمل حتى بدون الصور، لكنها ستظهر بشكل أفضل عند وضع صورك الحقيقية.
